## Commands

- `TimeForge: Start Tracking`: Manually start time tracking
- `TimeForge: Stop Tracking`: Pause time tracking
- `TimeForge: View Report`: Display time tracking statistics

## Requirements

- VS Code 1.60.0 or higher
